# PemrogramanLanjut_Inventarisasi
Repository proyek akhir kelas pemrograman lanjut

# Deskripsi : 
Program ini bertujuan untuk membantu proses inventarisasi pada toko retail agar sistem pencatatan stock product terjadi secara real time.

# Fitur - Fitur :
- Penyimpanan external (JSON file)
- Dual search engine (linear & Jump search)
- Kapabilitas sorting besar (Quick sort)
